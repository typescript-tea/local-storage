Effect manager for local storage

## Introduction

Use local storage in typescript-tea.

## How to use

Install:

```
yarn add @typescript-tea/local-storage
```

## How to develop

Node version >=12.6.0 is needed for development.

To execute the tests run `yarn test`.

## How to publish

```
yarn version --patch
yarn version --minor
yarn version --major
```
